# Changelog

## v0.1.0

Initial local release candidate for AgenticRouter / DevSpace Smart Router.

- Rule-based model routing across cheap, mid, and advanced DevSpace model tiers.
- DevSpace project catalog, routing rules, model aliases, profiles, fallbacks, and session stickiness.
- Golden evaluation dataset and evaluator.
- Dependency-free local web UI.
- Context Pack Builder and DevSpace Run Packet Generator.
- Outcome feedback logging.
- Local observability with sanitized traces and LangSmith-app-compatible file exports.
- Enterprise gateway template exports.
- Config Studio validation, import/export, and guarded add-project flow.
- Scenario Simulator with abstract cost/context savings.
- Stable DevSpace v1 integration API with advise, packet, shadow, and strict modes.
- Shadow Mode Analytics and rollout reporting.
- Pilot Readiness report, demo script, rollout plan, and demo scorecard.
- Release smoke test and port-finding web launcher.
